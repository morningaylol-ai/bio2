# BEL_MONARH OS

Легендарный Telegram-био сайт под Next.js 15.

## Запуск
```bash
npm i
npm run dev
```

## Деплой
- Залей на GitHub
- Подключи к Vercel
- Deploy

## Что внутри
- Preloader
- Кастомный курсор
- Золотые частицы
- 3D-сцена
- Терминал
- Dashboard
- Workflow
- Stack wall
- Final CTA
- Mobile responsive
- Tailwind + Framer Motion + GSAP + Three.js + Recharts
